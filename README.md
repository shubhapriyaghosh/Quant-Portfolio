# Quant Portfolio 2026

This repository contains five quant-finance portfolio projects:

1. Algorithmic Trading Backtester
2. Portfolio Optimization Engine
3. Market Risk Dashboard
4. Option Pricing Engine
5. Credit Risk Model

These projects are designed to show proof of skills in:

- Python
- Statistics
- Probability
- Financial markets
- Risk management
- Portfolio theory
- Derivatives
- Machine learning
- Dashboarding

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate      # Mac/Linux
.venv\Scripts\activate       # Windows

pip install -r requirements.txt
```

Run any project:

```bash
python project_1_algo_backtester/main.py
python project_2_portfolio_optimizer/main.py
python project_3_market_risk_dashboard/main.py
python project_4_option_pricing_engine/main.py
python project_5_credit_risk_model/main.py
```

Run dashboards:

```bash
streamlit run project_1_algo_backtester/dashboard_app.py
streamlit run project_2_portfolio_optimizer/dashboard_app.py
streamlit run project_3_market_risk_dashboard/dashboard_app.py
streamlit run project_4_option_pricing_engine/dashboard_app.py
streamlit run project_5_credit_risk_model/dashboard_app.py
```
